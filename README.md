# Bellum

Bellum is a premium AI exam-prep SaaS for medical and pre-med students.

It turns lecture PDFs into exam-grade MCQs with traps, explanations, weakness review, and ranked Deep Exam practice later.

## Current Phase

Phase 1 — Clickable Bellum Shell

This phase includes:

- Landing page
- White/blue academic brand direction
- Primary CTA: Generate Your First Quiz
- Dashboard shell
- Mock free trial and credit state
- Seeded demo quiz preview
- Generate placeholder
- Mistake Log placeholder
- Locked leaderboard teaser
- Clickable navigation
- No real AI APIs
- No real auth
- No real payments
- No database
